3. Growth of Functions

Created@Jun 2, 2020 1:37 AM
  • asymptotic efficiency: how the running time increases with the size of the input in the limit
  • Usually, an algorithm that is asymptotically more efficient will be the best choice for all but very small inputs

3.1 Asymptotic notation

  • Asymptotic tight bound Θ: Θ(g(n)) = { f(n) : there exist positive constants c1, c2, and n0 s.t 0 ≤ c1 g(n) ≤ f(n) ≤ c2 g(n) for all n ≥ n0} → "set of functions" ⇒ the function f(n) is equal to g(n) to within a constant factor, f(n) = Θ(g(n)) ※ assume every function used within Θ notation is asymptotically nonnegative
  • Asymptotic upper bound O: O(g(n)) = { f(n) : there exist positive constants c and n0 s.t 0 ≤ f(n) ≤ c g(n) for all n ≥ n0} - f(n) = Θ(g(n)) implies f(n) = O(g(n)) - when we use O to bound the worst-case running time, we have a bound on every input ↔ Θ notation
  • Asymptotic lower bound Ω: Ω(g(n)) = { f(n) : there exist positive constants c and n0 s.t 0 ≤ c g(n) ≤ f(n) for all n ≥ n0} - gives a lower bound on the best-case running time
  • Theorem 3.1: f(n) = Θ(g(n)) if and only if f(n) = O(g(n)) and f(n) = Ω(g(n))
  • Asymptotic non-tight upper bound o: o(g(n)) = { f(n) : for any positive constant c > 0, there exist a constants n0 > 0 s.t 0 ≤ f(n) < c g(n) for all n ≥ n0} - lim n→∞ f(n) / g(n) = 0
  • Asymptotic non-tight lower bound ω: ω(g(n)) = { f(n) : for any positive constant c > 0, there exist a constants n0 > 0 s.t 0 ≤ c g(n) < f(n) for all n ≥ n0} - lim n→∞ f(n) / g(n) = ∞

Comparing functions

  • Transitivity: f(n) = Θ(g(n)) and g(n) = Θ(h(n)) imply f(n) = Θ(h(n)) Θ, O, Ω, o, ω
  • Reflexivity: f(n) = Θ(f(n)) Θ, O, Ω
  • Symmetry: f(n) = Θ(g(n)) if and only if g(n) = Θ(f(n)) Θ
  • Transpose Symmetry: f(n) = O(g(n)) if and only if g(n) = Ω(f(n)) O, Ω, o, ω
  • ⚠️ Trichotomy does not hold (not all functions are asymptotically comparable, i.e. sin function)

3.2 Standard notations and common functions

  • any exponential function with a base strictly greater than 1 grows faster than any polynomial function: n^b = o(a^n)
  • any positive polynomial function grows faster than any polylogarithmic function: (lg n)^b = o (n^a)
  • n! = o(n^n) ; n! = ω(2^n) ; lg(n!) = Θ(n lg n)
