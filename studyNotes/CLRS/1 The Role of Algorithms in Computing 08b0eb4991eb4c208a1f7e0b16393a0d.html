1. The Role of Algorithms in Computing

Created@Jun 1, 2020 7:19 AM

1.1 Algorithms

  • algorithm: well-defined computational procedure that takes input (= instance) and produces output
  • correct: halts with the correct output for every input instance
  • data structure: a way to store and organize data in order to facilitate access and modifications
  • a problem is NP-complete if it is both in NP and NP-hard (no efficient solution is known) - NP: Nondeterministic Polynomial time

1.2 Algorithms as a technology

  • total system performance depends on choosing efficient algorithms as much as on choosing fast hardware
  • algorithms are at the core of most technologies used in contemporary computers
