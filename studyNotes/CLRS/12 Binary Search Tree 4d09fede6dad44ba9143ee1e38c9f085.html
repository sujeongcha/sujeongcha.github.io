  • Search tree can be used both as a dictionary and as a priority queue

12.1 What is a binary search tree?

  • binary-search-tree property: [left subtree] ≤x\leq x ≤x﻿, [right subtree] ≥x\geq x ≥x﻿
  • inorder tree walk: print out all the keys in sorted order by a simple recursive algorithm (O(n)O(n)O(n)﻿) - preorder tree walk: prints the root before the values in either subtree - postorder tree walk: prints the root after the values in its subtrees

12.2 Querying a binary search tree

Searching: O(h)

  • can be unrolled into a while loop → more efficient on most computer

Minimum and maximum: O(h)

Successor and predecessor: O(h)

12.3 Insertion and Deletion

Insertion: O(h)

Deletion: O(h)
